1：安装django-cors-headers
2：setting配置文件修改:
    MIDDLEWARE
        1：关于csrf的修改为：'corsheaders.middleware.CorsMiddleware',尽量加在靠前的位置
        最好在...CommonMiddleware的上方
        2：INSTALLED_APPS加corsheaders
        3：添加:CORS_ORIGIN_ALLOW_ALL = True # 白名单不启用
        4：添加：CORS_ORIGIN_WHITELIST = [] # 设置白名单
        5：添加：CORS_ALLOW_METHODS = (
                        'DELETE',
                        'GET',
                        'OPTIONS',
                        'PATCH',
                        'POST',
                        'PUT',
                        'VIEW',
                    )  # 预检请求可接受的方法
        6：添加：CORS_ALLOW_HEADERS = (
                        'XMLHttpRequest',
                        'X_FILENAME',
                        'accept-encoding',
                        'authorization',
                        'content-type',
                        'dnt',
                        'origin',
                        'user-agent',
                        'x-csrftoken',
                        'x-requested-with',
                        'Pragma',
                    ) # 预检请求可接受的头
        7：添加：CORS_ALLOW_CREDENTIALS = True  # 允许携带cookie
